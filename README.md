# yesplaymusic-plus-pkgbuild

This repository contains the `PKGBUILD` file and scripts for building and publishing the AUR package for **YesPlayMusic Plus** version 0.4.10.

## Contents

- **pre-PKGBUILD**: Template PKGBUILD file with placeholder for conflicts list
- **PKGBUILD**: The final build script used by `makepkg` (generated by prepare.py)
- **.SRCINFO**: Metadata file generated from `PKGBUILD`, necessary for AUR submissions
- **yesplaymusic-plus.desktop**: Desktop entry file for YesPlayMusic
- **yesplaymusic-plus.sh**: Launcher script for YesPlayMusic
- **prepare.py**: Script to fetch conflicts list from AUR and generate final PKGBUILD
- **publish_script.sh**: Script to automate the building and publishing process to the AUR

## Usage

### Building and Testing Locally

1. Install build dependencies:
   ```bash
   sudo pacman -S yarn nvm
   ```

2. Generate the final PKGBUILD:
   ```bash
   python prepare.py
   ```

3. Build and install the package:
   ```bash
   makepkg -si
   ```

### Publishing to AUR

Run the publish script:
```bash
./publish_script.sh
```

## About YesPlayMusic

YesPlayMusic is a beautiful and powerful music player that supports multiple music services. This repository specifically deals with version 0.4.10 for Arch Linux users who wish to install it via the AUR
